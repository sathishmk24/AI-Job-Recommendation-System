# AI Job Recommendation System

An AI-powered web application built using Python, Flask, SQLite, Scikit-Learn, and PyPDF2.

## Features

- Resume PDF Upload
- Skill Extraction
- AI Job Recommendations
- Resume Score Analyzer
- Admin Dashboard
- Add/Edit/Delete Jobs
- User Registration & Login
- Session Management

## Technologies Used

- Python
- Flask
- SQLite
- Scikit-Learn
- PyPDF2
- HTML
- Bootstrap

## Run Locally

pip install -r requirements.txt

python app.py
